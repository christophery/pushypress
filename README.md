#PushyPress

PushyPress is a responsive WordPress theme built with Bootstrap 3 featuring an off-canvas navigation menu (Pushy) included.

##Features

- Responsive
- Pushy
- Bootstrap 3
- Retina Ready

##Requirements

- WordPress 3.6

##Usage

1. Unzip and rename the directory from ```pushypress-master``` to ```pushypress```.

2. Place the ```pushypress``` directory in ```wp-content/themes```.

3. Activate it via your WordPress Dashboard.
